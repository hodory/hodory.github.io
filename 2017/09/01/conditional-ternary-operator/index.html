［PHP］삼항 연산자

Author  included in  category PHP
 2017-09-01  2017-09-01  41 words  One minute 

삼항연산자(?:)는 조건식 ? 참일때 실행할로직 : 거짓일때 실행할 로직; 과 같이쓴다. (if-else를 짧게 줄인것)

a 변수값이 있는지 검사한후 값이 있으면 foo값을 반환, 없으면 bar라는 값을 반환

ex)

<?php
$a = 'foo';
$b = $a ?: 'bar';
echo $b;    // foo
?>
