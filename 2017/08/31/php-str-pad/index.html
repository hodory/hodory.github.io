PHP에서 0001과 같이 번호 지정하기

<?php
$count = 1;
$count = str_pad($count,4,'0',STR_PAD_LEFT); //0001
?>

01이나 001 같이 사용할 때는 두번째 파라미터 값에 2(01) 또는 3(001)을 사용하면 된다.
