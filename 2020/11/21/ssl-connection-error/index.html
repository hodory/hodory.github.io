SSL Connection Error 발생

Contents

  • 해결방법

  • 참고 자료
[root@aaaaa ~]# curl -XGET {{링크}}
curl: (35) SSL connect error

원인

Curl 버전이 낮아서 발생하는 문제

해결방법

패키지를 업데이트 할 수 있을 때

$ yum update -y nss curl libcurl

패키지를 업데이트 할 수 없을 때

-1 을 파라미터로 넣어주면 해결 가능하다. (https://github.com/userify/shim/issues/25)

$ curl -XGET {{링크}} -k -1
